
# FastTest
基于Django的测试平台


## 特性
- 自定义测试库，可以支持接口测试，WebUI、AppUI测试，数据库测试等多种测试


## 使用方法

1. 下载或克隆当前项目
```shell
git clone https://github.com/hanzhichao/fasttest
```

2. 安装依赖

```shell
cd fasttest
pip install -r requirements.txt
```
3. 运行开发服务器

```shell
python manage.py runserver
```
4. 访问 http://localhost:8000/admin  用户名/密码 superhin/abc123
